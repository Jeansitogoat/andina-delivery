# Sonidos de notificación

Coloca aquí un archivo **new-order.mp3** (o renómbralo a `new-order.mp3`) con un sonido corto de 1–2 segundos para notificaciones de nuevo pedido o carrera asignada.

Si el archivo no existe, la app no falla; simplemente no se reproducirá sonido cuando llegue un pedido o carrera nueva.
