# Sonidos de notificación

**Importante:** Sin el archivo `new-order.mp3`, no habrá sonido en los paneles (central, restaurante, rider) cuando llegue un pedido o carrera nueva. La app no falla; simplemente no se reproduce ningún sonido.

Coloca aquí un archivo **new-order.mp3** (o renómbralo a `new-order.mp3`) con un sonido corto y audible de 1–2 segundos. Se usa en:

- Panel central: nuevo pedido esperando rider
- Panel restaurante: nuevo pedido entrante
- Panel rider: nueva carrera asignada

Puedes usar sonidos libres de derechos (por ejemplo, de freesound.org o similar) o grabar uno propio. El volumen se fija a 1.0 en la app para máxima audibilidad.
